# aws-autoscaling-loadbalancer
Implementation of AWS Auto Scaling with Load Balancer

## Introduction
During this Practical , I explored how **AWS Auto Scaling** works in combination with a **Load Balancer** to manage traffic and ensure high availability.  
I learned how to create and configure EC2 instances, launch templates, auto scaling groups, and load balancers, as well as how to test dynamic scaling in a practical scenario.
